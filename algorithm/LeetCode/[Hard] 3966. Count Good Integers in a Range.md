3966. Count Good Integers in a Range

You are given three integers `l`, `r` and `k`.

A number is considered **good** if the **absolute difference** between every pair of **adjacent** digits is at most `k`.

Return the number of good integers in the range `[l, r]` (inclusive).

The **absolute difference** between values `x` and `y` is defined as `abs(x - y)`.

 

**Example 1:**
```
Input: l = 10, r = 15, k = 1

Output: 3

Explanation:

The good integers in the range are 10, 11, and 12.
For 10, abs(1 - 0) = 1.
For 11, abs(1 - 1) = 0.
For 12, abs(1 - 2) = 1.
All these differences are at most k = 1. Thus, the answer is 3.
```

**Example 2:**
```
Input: l = 201, r = 204, k = 2

Output: 2

Explanation:

The good integers in the range are 201 and 202.
For 201, abs(2 - 0) = 2 and abs(0 - 1) = 1.
For 202, abs(2 - 0) = 2 and abs(0 - 2) = 2.
Thus, the answer is 2.
```

**Constraints:**

* `10 <= l <= r <= 10^15`
* `0 <= k <= 9`

**Solution 1: (Digit DP, Prefix Sum)**

__Intuition__
To count the number of valid integers in a given range [l,r], we can calculate the number of valid integers up to r and subtract the valid integers up to l−1. This narrows our task to solving a single function f(num, k) which finds all good integers ≤num.

Because the maximum constraint is 10 
15
 , checking every number linearly will result in a Time Limit Exceeded (TLE). However, the maximum length of the number as a string is only 16 digits. This makes Digit DP the perfect tool.

__Approach__
We build a recursive function get(s, n, i, prev, small, k) that builds numbers digit-by-digit from left to right.

State Variables:
i: The current index in the string we are filling.
prev: The previously placed digit (to enforce the k condition).
small: A boolean flag. If true, the current prefix is strictly smaller than the prefix of our target number, meaning we can place any valid digit up to 9. If false, we are bounded by the current digit of our target number.
Handling Numbers of Different Lengths: Instead of passing leading zeroes into our DP, we iterate over the starting length len.
If len == 0, we are building a number of the exact same length as our upper bound string s. The first digit can go up to s[0].
If len > 0, we are building a number shorter than s. The first digit can safely go up to 9, and the small flag is instantly true.
Memoization: Since the state space is tiny (16 positions × 10 possible previous digits × 2 boolean states), we cache the results in a 3D dp array to eliminate redundant calculations.
**Dry Run: f(15, 1)**
Let's find all good integers up to 15 with k=1.
The string is s = "15", length n = 2.

Loop 1: len = 0 (Building 2-digit numbers)

The max starting digit is mx = s[0] = 1.
We place prev = 1. Because 1==mx, small = false.
Call get(i=1, prev=1, small=false).
Since k = 1, the next digit can range from max(0, 1-1) = 0 to min(9, 1+1) = 2.
Because small == false, we are strictly bounded by s[1] = 5. Our upper bound remains min(2, 5) = 2.
Valid digits for the second position: 0, 1, 2.
The recursive calls reach the base case (i == n) and return 1 each.
Numbers formed: 10, 11, 12. (3 valid numbers).
Loop 2: len = 1 (Building 1-digit numbers)

The max starting digit here is 9, because a 1-digit number is automatically smaller than 15.
We iterate prev from 1 to 9.
Call get(i=2, prev, small=true) for each.
Since i == n, they instantly hit the base case and return 1.
Numbers formed: 1, 2, 3, 4, 5, 6, 7, 8, 9. (9 valid numbers).
Result: 3 + 9 = 12 good integers.

__Complexity Analysis__
Time Complexity: O(N⋅10⋅2⋅10), where N is the length of the string (up to 16). The states are bounded by the dp dimensions, and the inner loop iterates a maximum of 10 times. Overall time complexity is constant O(1).
Space Complexity: O(N⋅10⋅2) for the DP table and recursion stack space, which equates to O(1) auxiliary space.

```c++
class Solution {
    vector<vector<vector<long long>>>dp;
    long long get(string &s,int n,int i,int prev,bool small,int k){
        if(i==n)return 1;
        if(dp[i][prev][small]!=-1)return dp[i][prev][small];
        long long res=0;
        int start= max((int)0, prev-k);
        int end= min((int)9, prev+k);
        if(small==false)end= min(end,(int)(s[i]-'0'));
        
        for(int j=start;j<=end;j++){
            res+= get(s,n,i+1,j,small or (j< (s[i]-'0') ),k);
        }
        return dp[i][prev][small]=res;
    }
    long long f(long long num,long long k){
        dp.assign(16,vector<vector<long long>>(10,vector<long long>(2,-1)));

        string s= to_string(num);
        int n=s.size();
        long long res=0;
        for(int len=0;len<n;len++){
            int mx= (len==0?s[0]-'0':9);
            for(int prev=1;prev<=mx;prev++){
                if(len==0 and prev==mx){
                    res += get(s,n,len+1,prev,false,k);
                }
                else{
                    res += get(s,n,len+1,prev,true,k);
                }
            }
        }
        return res;
    }
public:
    long long goodIntegers(long long l, long long r, int k) {
        return f(r,k)-f(l-1,k);
    }
};
```
