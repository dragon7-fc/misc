3883. Count Non Decreasing Arrays With Given Digit Sums

You are given an integer array `digitSum` of length `n`.

An array `arr` of length `n` is considered valid if:

* * `0 <= arr[i] <= 5000`
* * it is **non-decreasing**.
* the **sum** of the digits of `arr[i]` equals `digitSum[i]`.

Return an integer denoting the number of **distinct valid arrays**. Since the answer may be large, return it modulo `10^9 + 7`.

An array is said to be **non-decreasing** if each element is greater than or equal to the previous element, if it exists.

 

**Example 1:**
```
Input: digitSum = [25,1]

Output: 6

Explanation:

Numbers whose sum of digits is 25 are 799, 889, 898, 979, 988, and 997.

The only number whose sum of digits is 1 that can appear after these values while keeping the array non-decreasing is 1000.

Thus, the valid arrays are [799, 1000], [889, 1000], [898, 1000], [979, 1000], [988, 1000], and [997, 1000].

Hence, the answer is 6.
```

**Example 2:**
```
Input: digitSum = [1]

Output: 4

Explanation:

The valid arrays are [1], [10], [100], and [1000].

Thus, the answer is 4.
```

**Example 3:**
```
Input: digitSum = [2,49,23]

Output: 0

Explanation:

There is no integer in the range [0, 5000] whose sum of digits is 49. Thus, the answer is 0.
```
 

**Constraints:**

* `1 <= digitSum.length <= 1000`
* `0 <= digitSum[i] <= 50`

# Submissions
---
**Solution 1: (Prefix Sum)**

__Intuition__
Precompute all numbers from 0 → 5000 grouped by their digit sum.

At position pos, we can only place numbers whose digit sum equals digitSum[pos].

If we place currNum at position pos, the previous number prevNum must satisfy:

prevNum ≤ currNum (non-decreasing condition)
sum_of_digits(prevNum) = digitSum[pos - 1]
So for each currNum, we need to count how many valid prevNum exist and how many ways they contribute.

__Approach__
Let:

DP[pos][currNum] = number of ways to form valid array till index pos ending at currNum
Base case:

DP[0][currNum] = 1 if digit sum matches digitSum[0], else 0
Naive transition:

for currNum:
    for prevNum in numbersWithDigitSum[digitSum[pos - 1]]:
        if(prevNum ≤ currNum):
            DP[pos][currNum] += DP[pos - 1][prevNum]
This loop is expensive (up to 5000 per state).
Optimization
__Observation:__

DP[pos][currNum] = sum of DP[pos - 1][prevNum] for all prevNum ≤ currNum
Precompute prefix sums on previous row:

prefixDP[x] = DP[pos - 1][0] + ... + DP[pos - 1][x]
For each currNum:

Find maximum prevNum ≤ currNum using binary search
Let it be max_prevNum
Transition becomes:

DP[pos][currNum] = prefixDP[max_prevNum]
Removes inner loop, makes transition O(1)

__Complexity__
Time:

O(n * 5000)
Space:

O(5000)

```
Runtime: 1360 ms, Beats 46.15%
Memory: 418.53 MB, Beats 7.69%
```
```c++
class Solution {
    int mod = 1e9 + 7;
    vector<int> numbersWithSum[55];

    void computesums(){
        for(int num = 0 ; num <= 5000 ; num++){
            int sum = 0;
            for(int curr = num ; curr > 0 ; curr /= 10){
                sum += curr % 10;
            }
            numbersWithSum[sum].push_back(num);
        }
    }
public:
    int countArrays(vector<int>& digitSum) {
        if(numbersWithSum[0].empty() == true) computesums();

        int n = digitSum.size();
        vector<int> prefixDP(5005 , 0);
        
        for(int pos = 0 ; pos < n ; pos++){
            vector<int> curr(5005 , 0);
            // try placing numbers whose sum of digits = digitsum[pos]
            for(int currNum : numbersWithSum[digitSum[pos]]){
                if(pos == 0){
                    curr[currNum] = 1;
                }
                else{
                    // find maximum PrevNumber with binary search
                    vector<int> &prevNumbers = numbersWithSum[digitSum[pos - 1]];
                    int maxIdx = upper_bound(prevNumbers.begin() , prevNumbers.end() , currNum) - prevNumbers.begin() - 1;
                    if(maxIdx >= 0) curr[currNum] = prefixDP[prevNumbers[maxIdx]];
                }
            }
            // compute prefix sum on DP[pos] and it will be used for DP[pos + 1]
            for(int num = 1 ; num <= 5000 ; num++) curr[num] = (curr[num] + curr[num - 1]) % mod;
            prefixDP = curr;
        }

        return prefixDP[5000];
    }
};
```
