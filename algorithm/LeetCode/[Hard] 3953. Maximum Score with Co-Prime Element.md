3953. Maximum Score with Co-Prime Element

You are given an integer array `nums` of length `n` and an integer `maxVal`.

You may change any element in nums to any positive integer less than or equal to `maxVal`. Each such change costs 1.

Two integers are **co-prime** if their **greatest common divisor (GCD)** is 1.

After all modifications, you must choose an index `i` such that, `nums[i]` is **co-prime** with every other element `nums[j]`.

Let:

* `selectedValue` be the final value of `nums[i]` after modifications.
* `modificationCost` be the total number of elements changed.
The score is defined as `score = selectedValue - modificationCost`.

Return the **maximum** possible score.

 

**Example 1:**
```
Input: nums = [3,4,6], maxVal = 5

Output: 4

Explanation:

Change nums[2] from 6 to 5, which costs 1. Choose nums[2] = 5, since it is co-prime with 3 and 4.

selectedValue = 5
modificationCost = 1
The score is 5 - 1 = 4
```

**Example 2:**
```
Input: nums = [1,2,3], maxVal = 4

Output: 3

Explanation:

No modifications are required. Choose nums[2] = 3, since it is co-prime with 1 and 2.

selectedValue = 3
modificationCost = 0
The score is 3 - 0 = 3
```

**Example 3:**
```
Input: nums = [2,2], maxVal = 1

Output: 1

Explanation:

Change nums[0] from 2 to 1, which costs 1. Choose nums[1] = 2, since it is co-prime with 1.

selectedValue = 2
modificationCost = 1
The score is 2 - 1 = 1
```

**Constraints:**

* `1 <= nums.length <= 10^5`
* `1 <= nums[i] <= 10^5`
* `1 <= maxVal <= 10^5`

# Submissions
---
**Solution 1: (Sieve + PIE + Bitmasks)**

__Intuition__
The core idea is to find a final target value and figure out how many elements in the array share a prime factor with it. These sharing elements are "conflicting" and must be changed, which represents our cost. Instead of blindly checking every element against our target, we can precalculate the prime factors and multiples beforehand. Furthermore, if we check candidates from the largest possible number down to the smallest, we can stop the search instantly the moment our current best score is higher than the candidate itself.

__Approach__
Precompute Data: First, record the frequencies of all numbers in the array. Then, use a Sieve to precompute the Smallest Prime Factor (SPF) for quick factorization, alongside an array mul that stores exactly how many numbers in our input are multiples of any given value.
Backward Loop (Early Exit): Iterate candidates backwards starting from the highest possible value down to 2. Since our score is candidate - cost (and cost is never negative), the score can never exceed the candidate's value. If our current max score (ans) is already greater than or equal to the candidate, we can safely break the loop and finish.
Count Conflicts: For each candidate, find its distinct prime factors. We use a standard bitmask to group these primes and figure out how many existing numbers in the array share at least one prime factor (this is the Principle of Inclusion-Exclusion). We add single primes, subtract pairs, add triplets, etc., to get an exact conflict count.
Calculate Cost: - If the candidate is already in the array, the cost is the number of conflicting elements plus the duplicates of the candidate we need to overwrite.
If the candidate is not in the array, we must spend 1 modification to create it, plus overwrite any remaining conflicts.

__Complexity__
Time complexity:
O(N+MlogM) where N is the length of nums and M is the maximum value. Building the frequency array takes O(N). The Sieve and multiples precomputation take O(MlogM). Evaluating the bitmasks is strictly bounded because a number ≤10 
5
  has at most 6 distinct prime factors, and the backward loop almost always breaks within the first few iterations (O(1) amortized).
Space complexity:
O(M) to store our flat freq, spf, and mul arrays. The prime factorization array inside the loop strictly uses a stack-allocated array of size 7, ensuring zero dynamic memory reallocation overhead.

```c++
class Solution {
public:
    int maxScore(vector<int>& nums, int maxVal) {
        int mx = maxVal;
        
        // 1. Find the absolute maximum value to size our arrays safely
        for (int x : nums) {
            if (x > mx) mx = x;
        }

        // 2. Flat frequency array
        vector<int> freq(mx + 1, 0);
        for (int x : nums) {
            freq[x]++;
        }

        vector<int> spf(mx + 1);
        for (int i = 0; i <= mx; i++) {
            spf[i] = i;
        }

        vector<int> mul(mx + 1, 0);
        
        // 3. Sieve to calculate Smallest Prime Factors and count Multiples
        for (int i = 2; i <= mx; i++) {
            if (spf[i] == i) {
                for (int j = i * 2; j <= mx; j += i) {
                    if (spf[j] == j) spf[j] = i;
                }
            }
            int m_sum = 0;
            for (int j = i; j <= mx; j += i) {
                m_sum += freq[j];
            }
            mul[i] = m_sum;
        }

        // Initialize baseline answer (Picking 1 is always free if it exists)
        int ans = freq[1] > 0 ? 1 : 0;
        
        // 4. Evaluate candidates backwards for the Early-Exit optimization
        for (int i = mx; i >= 2; i--) {
            
            // EARLY EXIT: If ans >= i, we can safely stop!
            if (i <= ans) break;
            
            if (freq[i] == 0 && i > maxVal) continue;

            int x = i;
            int fac[7]; // Max distinct primes <= 10^5 is 6. Stack array is fast.
            int cnt = 0;
            
            // Factorize the candidate
            while (x != 1) {
                int f = spf[x];
                fac[cnt++] = f;
                while (x % f == 0) x /= f;
            }

            int tot = 1 << cnt;
            int m = -freq[i]; // Start by excluding self-matches

            // 5. Standard Principle of Inclusion-Exclusion (PIE) Bitmask
            for (int j = 1; j < tot; j++) {
                int val = 1;
                
                // Normal 2^n loop to multiply active prime factors
                for (int k = 0; k < cnt; k++) {
                    if ((j >> k) & 1) {
                        val *= fac[k];
                    }
                }
                
                // If odd number of primes, ADD conflicts. If even, SUBTRACT.
                if (__builtin_popcount(j) & 1)
                    m += mul[val];
                else
                    m -= mul[val];
            }

            // 6. Calculate Final Cost and Update Score
            int cost = freq[i] > 0 ? (m + freq[i] - 1) : max(m, 1);
            if (i - cost > ans) {
                ans = i - cost;
            }
        }

        return ans;
    }
};
```
