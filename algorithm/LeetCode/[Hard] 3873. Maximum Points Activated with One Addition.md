3873. Maximum Points Activated with One Addition

You are given a 2D integer array `points`, where `points[i] = [xi, yi]` represents the coordinates of the `i`th point. All coordinates in points are distinct.

If a point is **activated**, then all points that have the **same** x-coordinate or y-coordinate become activated as well.

Activation continues until no additional points can be activated.

You may add one **additional** point at any integer coordinate `(x, y)` not already present in `points`. Activation begins by **activating** this newly added point.

Return an integer denoting the **maximum** number of points that can be activated, including the newly added point.

 

**Example 1:**
```
Input: points = [[1,1],[1,2],[2,2]]

Output: 4

Explanation:

Adding and activating a point such as (1, 3) causes activations:

(1, 3) shares x = 1 with (1, 1) and (1, 2) -> (1, 1) and (1, 2) become activated.
(1, 2) shares y = 2 with (2, 2) -> (2, 2) becomes activated.
Thus, the activated points are (1, 3), (1, 1), (1, 2), (2, 2), so 4 points in total. We can show this is the maximum activated.
```

**Example 2:**
```
Input: points = [[2,2],[1,1],[3,3]]

Output: 3

Explanation:

Adding and activating a point such as (1, 2) causes activations:

(1, 2) shares x = 1 with (1, 1) -> (1, 1) becomes activated.
(1, 2) shares y = 2 with (2, 2) -> (2, 2) becomes activated.
Thus, the activated points are (1, 2), (1, 1), (2, 2), so 3 points in total. We can show this is the maximum activated.
```

**Example 3:**
```
Input: points = [[2,3],[2,2],[1,1],[4,5]]

Output: 4

Explanation:

Adding and activating a point such as (2, 1) causes activations:

(2, 1) shares x = 2 with (2, 3) and (2, 2) -> (2, 3) and (2, 2) become activated.
(2, 1) shares y = 1 with (1, 1) -> (1, 1) becomes activated.
Thus, the activated points are (2, 1), (2, 3), (2, 2), (1, 1), so 4 points in total.
```
 

**Constraints:**

* `1 <= points.length <= 10^5`
`points[i] = [xi, yi]`
`-109 <= xi, yi <= 10^9`
`points` contains all distinct coordinates.

🧠 Intuition
The problem involves grouping points that share the same X or Y coordinates. This is a classic Connected Components problem. If point A shares an X-coordinate with point B, and point B shares a Y-coordinate with point C, then A, B, and C are all part of the same connected component.

By using Disjoint Set Union (DSU), we can efficiently merge these points into groups. The goal is to find the two largest independent groups. By "activating" an additional connecting point, we can merge these two largest components into one, plus the extra point itself.

🚀 Approach
DSU Initialization: Create parent and size arrays for n points. Each point starts as its own parent with a size of 1.
Coordinate Mapping: Use two HashMaps (xmap and ymap) to store the first point encountered for any given X or Y coordinate.
Union-Find: Iterate through the points. For each point:
If its X-coordinate has been seen before, union the current point with the stored point.
If its Y-coordinate has been seen before, union the current point with the stored point.
Path Compression & Union by Size: These optimizations ensure the DSU operations run in near-constant time (O(α(N))).
Find Largest Components: Iterate through the roots of the DSU to find the two largest sizes, max1 and max2.
Final Result: The answer is max1 + max2 + 1 (the two largest components plus one extra "bridge" point).
⚙️ Dry Run
Input: points = [[0,0], [0,1], [2,2]]

Point 0 (0,0): xmap[0]=0, ymap[0]=0.
Point 1 (0,1): Seen x=0. union(0,1). ymap[1]=1.
Point 2 (2,2): xmap[2]=2, ymap[2]=2. New component.
Sizes: Component A (Points 0,1) size = 2. Component B (Point 2) size = 1.
Result: 2 + 1 + 1 = 4.
📈 Complexity
Time complexity: O(N⋅α(N)) — Where α is the Inverse Ackermann function. The coordinate mapping and DSU operations are extremely efficient.
Space complexity: O(N) — To store the parent, size, and HashMaps for coordinates.

class Solution {
    vector<int> parent, sz;
    int find(int i) {
        if (parent[i] == i) return i;
        return parent[i] = find(parent[i]);
    }

    void unite(int i, int j) {
        int r1 = find(i), r2 = find(j);
        if (r1 != r2) {
            if (sz[r1] < sz[r2]) swap(r1, r2);
            parent[r2] = r1;
            sz[r1] += sz[r2];
        }
    }
public:
    int maxActivated(vector<vector<int>>& points) {
        int n = points.size();
        parent.resize(n); sz.assign(n, 1);
        iota(parent.begin(), parent.end(), 0);
        
        unordered_map<int, int> xmap, ymap;
        for (int i = 0; i < n; i++) {
            if (xmap.count(points[i][0])) unite(i, xmap[points[i][0]]);
            else xmap[points[i][0]] = i;
            if (ymap.count(points[i][1])) unite(i, ymap[points[i][1]]);
            else ymap[points[i][1]] = i;
        }

        int m1 = 0, m2 = 0;
        for (int i = 0; i < n; i++) {
            if (parent[i] == i) {
                if (sz[i] > m1) { m2 = m1; m1 = sz[i]; }
                else if (sz[i] > m2) { m2 = sz[i]; }
            }
        }
        return m1 + m2 + 1;
    }
};
