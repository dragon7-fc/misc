3956. Maximum Sum of M Non-Overlapping Subarrays I

You are given an integer array `nums` of length `n`, and three integers `m`, `l`, and `r`.

Your task is to select at least one and at most `m` **non-overlapping subarrays** from `nums` such that:

* Each selected **subarray** has a length between `[l, r]` (inclusive).
* The total sum of all selected **subarrays** is **maximized**.

Return the **maximum** total sum you can achieve.

 

**Example 1:**
```
Input: nums = [4,1,-5,2], m = 2, l = 1, r = 3

Output: 7

Explanation:

One optimal strategy is to:

Select the subarray [4, 1] with sum 4 + 1 = 5 and the subarray [2] with sum 2. Both subarrays have length between [l, r].
The total sum of these subarrays is 5 + 2 = 7, which is the maximum achievable sum with at most m = 2 subarrays.
```

**Example 2:**
```
Input: nums = [1,0,3,4], m = 2, l = 1, r = 2

Output: 8

Explanation:

One optimal strategy is to:

Select the subarray [1] with sum 1 and the subarray [3, 4] with sum 3 + 4 = 7. Both subarrays have length between [l, r].
The total sum of these subarrays is 1 + 7 = 8, which is the maximum achievable sum with at most m = 2 subarrays.
```

**Example 3:**
```
Input: nums = [-1,7,-4], m = 1, l = 2, r = 3

Output: 6

Explanation:

Select the subarray [-1, 7] from nums which has length between [l, r].
The total sum of this subarray is -1 + 7 = 6, which is the maximum achievable sum with at most m = 1 subarray.
```

**Example 4:**
```
Input: nums = [-3,-4,-1], m = 2, l = 1, r = 2

Output: -1

Explanation:

All subarrays of nums have negative sums. The optimal strategy is to select the subarray [-1], which has length between [l, r].
The total sum of this subarray is -1, which is the maximum achievable sum with at most m = 2 subarrays.
```

**Constraints:**

* `1 <= n == nums.length <= 1000`
* `-10^9 <= nums[i] <= 10^9`
* ``1 <= m <= n``
* ``1 <= l <= r <= n``

**Solution 1: (DP Bottom-Up, using prefix sum and monotonic dec queue, knapsack)**

dp[k][i] =    max    (dp[k][i-1], max{dp[k-1][j] + P[i] - P[j])
          i-r<=j<=i-l
                                  ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
                                  max(dp[k-1][j] - P[j]) + P[i]
                                   j
                                  ^^^^^^^^^^^^^^^^^^^^^^
                                       mono dec deque

```

Memory: 67.19 MB, Beats 13.33%
```
```c++
class Solution {
public:
    long long maximumSum(vector<int>& nums, int m, int l, int r) {
        int n = nums.size();
        
        // Use long long for tracking sums to prevent integer overflow errors
        const long long INF = -1e18; 
        
        // Step 1: Precompute prefix sums
        std::vector<long long> P(n + 1, 0);
        for (int i = 0; i < n; ++i) {
            P[i + 1] = P[i] + nums[i];
        }
        
        // Base arrays for space-optimized DP (only tracking current and previous rows)
        std::vector<long long> prev_dp(n + 1, 0); // Corresponds to k = 0 (0 subarrays = 0 sum)
        std::vector<long long> curr_dp(n + 1, INF);
        
        long long global_max = INF;
        
        // Step 2: Populate the DP row by row
        for (int k = 1; k <= m; ++k) {
            // Re-initialize current row to negative infinity
            std::fill(curr_dp.begin(), curr_dp.end(), INF);
            
            // Monotonic deque stores indices 'j' of the previous row's configurations
            std::deque<int> dq;
            
            for (int i = 1; i <= n; ++i) {
                // 1. Maintain sliding window: Add the newest valid option j = i - l
                int new_j = i - l;
                if (new_j >= 0 && prev_dp[new_j] != INF) {
                    long long new_val = prev_dp[new_j] - P[new_j];
                    
                    // Maintain monotonic decreasing order based on (prev_dp[j] - P[j])
                    while (!dq.empty() && (prev_dp[dq.back()] - P[dq.back()]) <= new_val) {
                        dq.pop_back();
                    }
                    dq.push_back(new_j);
                }
                
                // 2. Maintain sliding window: Remove expired indices j < i - r
                while (!dq.empty() && dq.front() < i - r) {
                    dq.pop_front();
                }
                
                // 3. Base choice: Carry forward the best sum without picking a subarray ending at i-1
                curr_dp[i] = curr_dp[i - 1];
                
                // 4. If the deque is not empty, the front contains the optimal starting index 'best_j'
                if (!dq.empty()) {
                    int best_j = dq.front();
                    long long current_sum = prev_dp[best_j] + (P[i] - P[best_j]);
                    if (current_sum > curr_dp[i]) {
                        curr_dp[i] = current_sum;
                    }
                }
            }
            
            // Capture the highest sum achieved across any valid count of subarrays up to k
            global_max = std::max(global_max, curr_dp[n]);
            
            // Move current row configurations to previous row for the next iteration step
            prev_dp = curr_dp;
        }
        
        return global_max;
    }
};
```
