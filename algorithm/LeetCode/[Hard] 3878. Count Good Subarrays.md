3878. Count Good Subarrays

You are given an integer array `nums`.

A **subarray** is called **good** if the **bitwise OR** of all its elements is equal to **at least one** element present in that subarray.

Return the number of good subarrays in `nums`.

Here, the bitwise OR of two integers `a` and `b` is denoted by `a | b`.

 

**Example 1:**
```
Input: nums = [4,2,3]

Output: 4

Explanation:

The subarrays of nums are:

Subarray  | Bitwise OR     | Present in Subarray
[4]         4 = 4            Yes
[2]         2 = 2            Yes
[3]	        3 = 3            Yes
[4, 2]      4 | 2 = 6        No
[2, 3]      2 | 3 = 3        Yes
[4, 2, 3]   4 | 2 | 3 = 7    No
Thus, the good subarrays of nums are [4], [2], [3] and [2, 3]. Thus, the answer is 4.
```

**Example 2:**
```
Input: nums = [1,3,1]

Output: 6

Explanation:

Any subarray of nums containing 3 has bitwise OR equal to 3, and subarrays containing only 1 have bitwise OR equal to 1.

In both cases, the result is present in the subarray, so all subarrays are good, and the answer is 6.
```
 

**Constraints:**

* `1 <= nums.length <= 10^5`
* `0 <= nums[i] <= 10^9`

# Submissions
---
**Solution 1: (left right)**

__Intuition & Approach__
For a subarray to have a bitwise OR exactly equal to an element nums[i] inside it, no other element in that subarray can have a 1 bit where nums[i] has a 0 bit.

Find the Boundaries: For every element nums[i], we expand to the left and right as far as possible. We stop the moment we hit a number that has a 1 in a bit position where nums[i] has a 0. This gives us the maximum valid range [L, R].
Handle Duplicates: If the same value appears multiple times in the array, we might count the same subarray twice. To prevent this, we force the left boundary L to start strictly after the last time we saw nums[i].
Count: Any subarray that starts between [L, i] and ends between [i, R] is perfectly bounded. The number of such subarrays is (i - L + 1) * (R - i + 1).
__Complexity__
Time complexity: O(31×N)≈O(N). We sweep the array to find bounds.
Space complexity: O(N), for the boundary arrays and the hash map.

```
Runtime: 467 ms, Beats 70.24%
Memory: 198.12 MB, Beats 73.81%
```
```c++
class Solution {
public:
    long long countGoodSubarrays(vector<int>& nums) {
        int n = nums.size();
        
        vector<int> prev_one(31, -1), next_one(31, n);
        vector<int> L(n, 0), R(n, n - 1);
        
        // Find the furthest left we can go without adding new bits
        for (int i = 0; i < n; i++) {
            for (int bit = 0; bit < 31; bit++) {
                if ((nums[i] & (1 << bit)) == 0) {
                    L[i] = max(L[i], prev_one[bit] + 1);
                } else {
                    prev_one[bit] = i;
                }
            }
        }
        
        // Find the furthest right we can go without adding new bits
        for (int i = n - 1; i >= 0; i--) {
            for (int bit = 0; bit < 31; bit++) {
                if ((nums[i] & (1 << bit)) == 0) {
                    R[i] = min(R[i], next_one[bit] - 1);
                } else {
                    next_one[bit] = i;
                }
            }
        }
        
        long long ans = 0;
        unordered_map<int, int> last_idx;
        
        // Calculate valid subarrays anchored at 'i'
        for (int i = 0; i < n; i++) {
            int l = L[i], r = R[i];
            
            // Prevent duplicate counting for the same OR maximum
            if (last_idx.count(nums[i])) {
                l = max(l, last_idx[nums[i]] + 1);
            }
            last_idx[nums[i]] = i;
            
            ans += 1LL * (i - l + 1) * (r - i + 1);
        }
        
        return ans;
    }
};
```
