3886. Sum of Sortable Integers

You are given an integer array `nums` of length `n`.

An integer `k` is called **sortable** if `k` **divides** `n` and you can sort `nums` in **non-decreasing** order by sequentially performing the following operations:

* Partition `nums` into **consecutive subarrays** of length `k`.
* **Cyclically rotate each subarray independently** any number of times to the left or to the right.

Return an integer denoting the sum of all possible sortable integers `k`.

A **subarray** is a contiguous **non-empty** sequence of elements within an array.
 

**Example 1:**
```
Input: nums = [3,1,2]

Output: 3

Explanation:

For n = 3, possible divisors are 1 and 3.
For k = 1: each subarray has one element. No rotation can sort the array.
For k = 3: the single subarray [3, 1, 2] can be rotated once to produce [1, 2, 3], which is sorted.
Only k = 3 is sortable. Hence, the answer is 3.
```

**Example 2:**
```
Input: nums = [7,6,5]

Output: 0

Explanation:

For n = 3, possible divisors are 1 and 3.
For k = 1: each subarray has one element. No rotation can sort the array.
For k = 3: the single subarray [7, 6, 5] cannot be rotated into non-decreasing order.
No k is sortable. Hence, the answer is 0.
```

**Example 3:**
```
Input: nums = [5,8]

Output: 3

Explanation:

For n = 2, possible divisors are 1 and 2.
Since [5, 8] is already sorted, every divisor is sortable. Hence, the answer is 1 + 2 = 3.
```

**Constraints:**

* `1 <= n == nums.length <= 10^5`
* `1 <= nums[i] <= 10^5`

**Solution 1: (Brute Force)**

__Intuition__
We are allowed to split the array into blocks of size k and cyclically rotate each block independently.

__Key observations:__

A cyclic rotation does not change the relative order pattern, it only shifts it.
For a block to be rearranged into non-decreasing order using rotations, it must already be a rotated sorted array.
A rotated sorted array has:

At most one "drop" point (i.e., where nums[i] > nums[i+1])
If such a drop exists, then the last element must be ≤ first element (valid rotation structure)
Additionally:

Since blocks remain in order, the maximum of the previous block must be ≤ minimum of the next block, otherwise global sorting is impossible.
Approach
We try all possible values of k such that k | n.

For each k, we:

Partition the array into subarrays of length k.

For each subarray:

Check if it has at most one drop point
If a drop exists, ensure it's a valid rotated sorted array
Ensure all elements in the block are ≥ previous block’s maximum
If all blocks satisfy the conditions, then k is valid.

Key checks inside each block:
Count number of nums[j-1] > nums[j]
If more than 1 → invalid
If exactly 1 → ensure last <= first
Track maximum of current block to validate next block
__Complexity__
Let (d(n)) is the number of divisors of n
d(n) ≈ √(n) on worst case
d(n) ≈ log(n) on average for large numbers

Time complexity: O(n √(n))

Space complexity: O(1)

```c++
class Solution {
    bool check(vector<int> &nums, int len) {
        int n = nums.size(), minn = 0;
        
        for (int i = 0; i < n; i += len) {
            if (nums[i] < minn) return false;
            
            int maxx = nums[i];
            bool point = false;
            
            for (int j = i + 1; j < i + len; j++) {
                if (nums[j] < minn) return false;
                
                if (nums[j - 1] > nums[j]) {
                    if (point) return false;
                    point = true;
                }
                
                maxx = max(maxx, nums[j]);
            }
            
            // If rotated, last must be <= first
            if (point) {
                if (nums[i] < nums[i + len - 1]) return false;
            }
            
            minn = maxx;
        }
        
        return true;
    }
public:
    int sortableIntegers(vector<int>& nums) {
        int n = nums.size(), ans = 0;
        
        for (int k = 1; k <= n; k++) {
            if (n % k != 0) continue;
            if (check(nums, k)) ans += k;
        }
        
        return ans;
    }
};
```
